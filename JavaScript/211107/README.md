# Hoisting(호이스팅) -> var, function
    - 아래 있는 선언을(만) 끌어올리다.
    - 함수를 예를 들면 함수 선언 -> 함수 호출이 일반적이지만 함수 호출 -> 함수 선언도 가능하다.
    - let은 호이스팅이 적용이 되지 않음.

# 자료형(Data Types)
    - 변수가 가지는 고정 타입이 없다.
    - 하지만 타입이 없는 것은 아니다.

# 데이터 타입
    - 기본 타입(Primitive values)
      - Boolean
      - Null
      - Undefined
      - Number
      - String
      - Symbol (ECMAScript 6에 추가됨) - > new로 생성 불가
    - 객체(Objects)
