# 211005
    - 문자 콘텐츠 : ol, ul, li
    - Ol 태그 : 숫자로 표현
      - type="1" -> 숫자로 표현
      - type="a" -> a,b,c 순서로 표현
      - type="A" -> A,B,C 순서로 표현
      - type="i" -> 로마 소문자 순서로 표현
      - type="I" -> 로마 대문자 순서로 표현
      - reversed -> 역순으로 표현 ex) 3,2,1
      - start="4" -> 지정 숫자부터 순서로 목록 표현 ex) 4,5,6
    - ul 태그 : ο 으로 표현
    - li 태그
      - 속성 value = 항목의 순서를 설정, 값 = 숫자, 특징 = 이하 항목들의 순서가 다시 지정됨
