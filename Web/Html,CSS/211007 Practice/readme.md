# dl,dt,dd 태그
- 용어(dt)와 정의(dd) 쌍들의 영역(dl)을 설정.
- dl은 dd,dt만을 포함해야 함.
- 키(key)/값(value) 형태를 표시할 때 유용.

# p 태그
- 하나의 문단을 설정.
- 일반적으로 정보통신보조기기 등은 다음 문단(p)으로 넘어갈 수 있는 단축키를 제공함. -> 웹접근성
- p {display : block; }

# hr 태그
- 문단의 분리(주제에 의한)를 위해 설정
- 대부분의 경우 수평선(border)으로 표시(표현적 관점)되나 의미적 관점으로만 사용해야 함.
- hr {display : block; }
